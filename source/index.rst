Welcome to ez-publish-japan-community!
======================================

eZ Publish は 100% オープンソースの Web CMS です。

eZ Publish 日本ユーザ会では、本家 eZ Publish を fork して、日本版 eZ Publish を開発・メンテナンスしています。

現在は github 上にリポジトリを持って、google グループ上で議論をしてます。

* `github <http://github.com/ez-publish-japan-community/ezpublish>`_
* `google group <http://groups.google.co.jp/group/ez-publish-japan>`_
